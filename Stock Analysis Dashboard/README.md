# Stock Analysis Dashboard

A comprehensive stock analysis dashboard built with Streamlit, providing real-time stock data visualization and technical indicators.

## Features

- Real-time stock data fetching via Yahoo Finance
- Interactive candlestick, line, and OHLC charts
- Technical indicators:
  - Simple Moving Average (SMA)
  - Exponential Moving Average (EMA)
  - Relative Strength Index (RSI)
- Performance metrics and statistics
- CSV data export functionality
- Responsive and intuitive UI
